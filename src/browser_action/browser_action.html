Symbol How to use
- When you put a dash before a word, it will exclude all posts with titles containting that word.
Example: gtx 1080 -MIR
d: This will only look for posts that are from the given domain name(s).
Example: monitor d:amazon.com d:bestbuy.com
" This will search for exact words in the quotes.
Example: "i5 6500"
* This is used as a wildcard character. When you put in inside a word, it will search for all instances of the word with the wildcard character replaced by a space, dash, and no wildcard character. a dash.
Example: GTX*1080
